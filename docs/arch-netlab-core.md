# Arch NetLab Core

## Purpose

KTX NetLab is a practical Linux-based networking lab built on a repurposed Dell laptop running Arch Linux. It is used for hands-on network diagnostics, Linux administration, SSH management, service monitoring, Docker networking, and future dashboard development.

## Current role

The Arch node acts as a dedicated lab machine on the local network. It is managed remotely from the Debian workstation using SSH.

## Network identity

- Hostname: ktx-netlab
- Active interface: enp19s0
- IP address: 192.168.254.160
- Gateway: 192.168.254.254
- DNS: 192.168.254.254

## Core services

- NetworkManager for network management
- SSH for remote administration
- Docker for container and virtual networking practice
- Caddy for future local web services
- LightDM/XFCE for lightweight graphical access when needed

## Lab value

This setup proves hands-on skill in Linux installation, remote administration, network troubleshooting, service management, Git documentation, and practical infrastructure building.
