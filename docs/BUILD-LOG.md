# KTX NetLab Build Log

## Goal

Repurpose an old Dell i3 laptop with 6GB RAM into a lightweight Arch Linux network lab workstation with XFCE, SSH, networking tools, and a custom dashboard.

## Build checkpoints

- [ ] Boot Arch live ISO
- [ ] Enable SSH in live environment
- [ ] Audit hardware
- [ ] Confirm boot mode
- [ ] Plan partitioning
- [ ] Install Arch base
- [ ] First boot
- [ ] Enable SSH
- [ ] Install XFCE
- [ ] Install network tools
- [ ] Start Cisco lab notes
- [ ] Build dashboard MVP
