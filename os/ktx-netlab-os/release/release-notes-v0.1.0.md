# KTX NetLab OS v0.1.0 Release Notes

## Summary

KTX NetLab OS v0.1.0 is the first installed operating environment build on top of Arch Linux.

It turns a repurposed Dell Inspiron N5040 into a practical network lab, IT support toolkit, portfolio machine, and future router lab foundation.

## Added

- KTX OS identity
- KTX command center
- KTX desktop session
- XFCE/xfwm4 appliance-style desktop layer
- KTX launchers
- KTX shell prompt
- KTX terminal profile
- KTX workspace folders
- KTX app defaults
- KTX fastfetch identity
- KTX doctor command
- KTX build manifest

## Not included yet

- Real router mode
- USB Ethernet LAN mode
- nftables NAT profile
- dnsmasq DHCP profile
- archiso ISO profile

## Status

Usable as a local installed network lab operating environment.
