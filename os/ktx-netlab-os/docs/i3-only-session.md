# KTX NetLab OS i3-only Session Policy

KTX NetLab OS now defaults to the KTX i3 Lab session.

## Active session

- KTX i3 Lab

## Policy

The visual shell target is i3-only for the main KTX NetLab OS experience.

XFCE is no longer the intended visible shell. i3 provides the clean enterprise rogue tiling workflow.

## Safety

This change affects only the graphical login/session layer.

It does not enable router mode, NAT, DHCP, packet forwarding, or firewall routing behavior.
